# Название

Оптимизация расписания лекций

# Описание

Оптимизация расписания лекций
В университете есть лекционный зал и набор лекций. Каждая лекция имеет:
- Время начала `start`
- Время окончания `end`
- Количество слушателей `students`
- Приоритет `prioroty` (1-высокий, 2-средний, 3-низкий)

Лекции не могут пересекаться по времени. Необходимо составить расписание, которое максимизирует:
1. Общее количество слушателей
2. Приоритет лекций (чем выше приоритет, тем меньше число)

**Целевая функция**: суммарное значение `(students * (4 - priority))` для выбранных лекций.

# Краткая теория

Математическая формулировка задачи

## Формальная постановка

### **Множества и индексы**

- $i, j \in \{1, 2, \dots, n\}$ — индексы лекций
- $L = \{l_1, l_2, \dots, l_n\}$ — множество всех лекций

### **Параметры**

Для каждой лекции $l_i$ заданы:

| Параметр | Обозначение | Описание |
|----------|-------------|-----------|
| Время начала | $s_i$ | Start time |
| Время окончания | $e_i$ | End time |
| Количество студентов | $stu_i$ | Number of students |
| Приоритет | $p_i \in \{1, 2, 3\}$ | Priority level |

### **Переменные решения**

$$
x_i = \begin{cases} 
1, & \text{если лекция } l_i \text{ выбрана в расписание} \\
0, & \text{в противном случае}
\end{cases}
$$

## Целевая функция

**Максимизировать суммарную ценность расписания:**

$$
\max Z = \sum_{i=1}^n \left[ x_i \cdot stu_i \cdot (4 - p_i) \right]
$$

### **Компоненты целевой функции:**

- $stu_i$ — вклад количества студентов
- $(4 - p_i)$ — весовой коэффициент приоритета:
  - Приоритет 1: вес = 3
  - Приоритет 2: вес = 2  
  - Приоритет 3: вес = 1

## ⛓️ Ограничения

### **1. Непересекаемость лекций**

Для каждой пары лекций $(i, j)$ с пересекающимися интервалами:

$$
x_i + x_j \leq 1, \quad \forall i,j: [s_i, e_i] \cap [s_j, e_j] \neq \emptyset
$$

**Альтернативная формулировка через временные слоты:**
$$
\sum_{\substack{i \\ s_i \leq t < e_i}} x_i \leq 1, \quad \forall t \in T
$$

### **2. Бинарность переменных**

$$
x_i \in \{0, 1\}, \quad \forall i \in \{1, 2, \dots, n\}
$$
