# Название

Особый порядок

# Описание

Дан массив целых чисел `nums`. Отсортируйте массив в **особом порядке**, удовлетворяющем следующим условиям:

1. **Все чётные числа** должны располагаться **перед** всеми нечётными числами
2. **Среди чётных чисел** должна быть сортировка по **убыванию** (от большего к меньшему)
3. **Среди нечётных чисел** должна быть сортировка по **возрастанию** (от меньшего к большему)

# Краткая теория





### 1. Кастомная сортировка 
В Java для кастомной сортировки используются:
- **Comparator** интерфейс
- **Lambda expressions** (Java 8+)
- **Arrays.sort()** или **Collections.sort()**

### 2. Компаратор
**Компаратор** - функциональный интерфейс, который определяет порядок двух элементов:
- Возвращает **отрицательное** число, если первый элемент должен идти перед вторым
- Возвращает **положительное** число, если второй элемент должен идти перед первым
- Возвращает **0**, если элементы равны

## Ключевые идеи решения

### Стратегия : Единый компаратор с лямбдой

```java
Arrays.sort(nums, (a, b) -> {
    boolean aEven = (a % 2 == 0);
    boolean bEven = (b % 2 == 0);
    
    if (aEven && bEven) {
        return b - a; // четные по убыванию
    } else if (!aEven && !bEven) {
        return a - b; // нечетные по возрастанию
    } else if (aEven) {
        return -1;    // четное перед нечетным
    } else {
        return 1;     // нечетное после четного
    }
});
