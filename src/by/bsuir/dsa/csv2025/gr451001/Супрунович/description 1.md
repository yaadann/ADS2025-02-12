# Название

BST: Поиск в допуске

# Описание

Поиск с допуском в BST
Реализуйте функцию для поиска в бинарном дереве поиска (BST) значения, наиболее близкого к заданному целевому значению, с учётом допустимой максимальной разницы. Если такого значения не найдено - вернуть "NO". При наличии двух подходящих чисел вывести наименьшее

Ограничения:
- Node.value > 0

# Краткая теория

Деревья поиска
Деревья поиска — это структуры данных, позволяющие эффективно хранить и обрабатывать упорядоченные данные. Наиболее распространенный тип — бинарные деревья поиска (BST).

## Основные свойства BST

Для каждого узла в BST выполняются условия:
- Все значения в **левом** поддереве **меньше или равны** значению узла
- Все значения в **правом** поддереве **больше или равны** значению узла

Это свойство позволяет выполнять поиск, вставку и удаление элементов в среднем за **O(log n)** времени.

## Балансировка

В худшем случае (вырожденное дерево) сложность операций достигает **O(n)**. Для решения этой проблемы существуют **сбалансированные** деревья:
- **AVL-деревья** — строгая балансировка (разница высот поддеревьев ≤ 1)
- **Красно-черные деревья** — менее строгая, но более эффективная балансировка
- **B-деревья** — оптимизированы для работы с диском
