# Название

Проверка бинарного дерева поиска

# Описание

Проверка бинарного дерева поиска

Имея бинарное дерево, *определить, является ли оно корректным бинарным деревом поиска*.

## Входные данные

Длина n массива ввода, n узлов дерева, которые могут быть целым числом или *null*. Узлы задаются построчно: первый - корень, второй и третий - его левый и правый потомки, 4-7 - их потомки и т.д. 

## Выходные данные

*true*, если дерево является двоичным деревом поиска, *false*, если не является

## Примеры

> 3 2 1 3
>  true

> 5 4 1 6 null null 5 7
> false

# Краткая теория

Деревья

Дерево — одна из наиболее распространенных структур данных в программировании.

Деревья состоят из набора вершин (узлов) и ориентированных рёбер (ссылок) между ними. Вершины связаны таким образом, что от какой-то одной вершины, называемой корневой, можно дойти до всех остальных единственным способом.

## Бинарные деревья поиска
Бинарное дерево поиска (англ. binary search tree, BST) — дерево, для которого выполняются следующие свойства:

- У каждой вершины не более двух детей.
- У всех вершин левого поддерева вершины значения не больше, чем значение самой вершины
- У всех вершин правого поддерева вершины значения не меньше, чем значение самой вершины
- Оба поддерева — левое и правое — являются двоичными деревьями поиска.
