# Описание

Операции в модулярной арифметике

Реализуйте программу для вычисления различных операций в модулярной арифметике.

Программа должна поддерживать:
- Базовые арифметические операции по модулю
- Возведение в степень по модулю
- Вычисление обратного элемента
- Решение систем линейных сравнений

## Формат ввода
Входные данные подаются в различных форматах:
1. `a + b mod m`
2. `a * b mod m` 
3. `a ^ b mod m`
4. `a ^-1 mod m` (обратный элемент)
5. `x ≡ a mod b, x ≡ c mod d, ...` (система сравнений)
6. Простые арифметические выражения

## Формат вывода
Для каждого входного выражения выведите результат вычисления.

# Краткая теория

## Модулярная арифметика

### Определение
Для целых чисел $a$, $b$ и $m > 0$:
$$a \equiv b \pmod{m} \iff m \mid (a - b)$$

### Основные свойства
1. **Сложение**: $(a + b) \bmod m = ((a \bmod m) + (b \bmod m)) \bmod m$
2. **Умножение**: $(a \cdot b) \bmod m = ((a \bmod m) \cdot (b \bmod m)) \bmod m$
3. **Возведение в степень**: $a^b \bmod m$ вычисляется алгоритмом быстрого возведения

### Обратный элемент
Обратный элемент $a^{-1} \pmod{m}$ существует тогда и только тогда, когда $\gcd(a, m) = 1$

### Китайская теорема об остатках
Для попарно взаимно простых модулей $m_1, m_2, \dots, m_k$ система:
$$
\begin{cases}
x \equiv a_1 \pmod{m_1} \\
x \equiv a_2 \pmod{m_2} \\
\vdots \\
x \equiv a_k \pmod{m_k}
\end{cases}
$$
имеет единственное решение по модулю $M = m_1 \cdot m_2 \cdot \dots \cdot m_k$
