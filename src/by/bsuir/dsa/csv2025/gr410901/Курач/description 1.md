# Название

Поиск в бинарном дереве поиска

# Описание

Поиск в бинарном дереве поиска
Реализуйте функцию поиска элемента в бинарном дереве поиска. Функция должна возвращать true, если элемент найден, и false в противном случае.
Входные данные:
Указатель на корень дерева
Целевое значение для поиска
Выходные данные:
Булево значение: true если элемент найден, false если нет

# Краткая теория

Бинарные деревья поиска
Бинарное дерево поиска (BST) — структура данных, обладающая следующими свойствами:
У каждой вершины не более двух детей
Все вершины обладают ключами с определённой операцией сравнения
У всех вершин левого поддерева вершины $v$ ключи не больше, чем ключ $v$
У всех вершин правого поддерева вершины $v$ ключи больше, чем ключ $v$
Оба поддерева являются бинарными деревьями поиска
Алгоритм поиска:
Если текущая вершина пуста — элемент не найден
Если ключ текущей вершины равен искомому — элемент найден
Если искомый ключ меньше ключа текущей вершины — ищем в левом поддереве
Если искомый ключ больше ключа текущей вершины — ищем в правом поддереве
Сложность: O(h), где h — высота дерева
