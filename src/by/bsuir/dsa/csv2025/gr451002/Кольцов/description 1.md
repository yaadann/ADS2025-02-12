# Название

Битовое представление чисел

# Описание

Битовое представление чисел 

Необходимо реализовать класс BinaryString для работы с числами в двоичном формате. 
Необходимо реализовать:
* Конструкторы:
    * Без параметров: экземпляр объекта должен инициализироваться 0-ём
    * С параметром типа Double: экземпляр объекта должен инициализироваться значением аргумента

* Методы:
    * BinaryString add(BinaryString value): Сложение двух двоичных чисел (результат сохраняется в вызывающий объект)
    * BinaryString sub(BinaryString value): Разность двух двоичных чисел (результат сохраняется в вызывающий объект)
    * BinaryString mul(BinaryString value): Произведение двух двоичных чисел (результат сохраняется в вызывающий объект)
    * BinaryString div(BinaryString value): Произведение двух двоичных чисел (результат сохраняется в вызывающий объект)
    * double getDoubleValue(): получение десятичного представления значения вызывающего объекта
    * String getValue(): получение строки двоичного значения вызывающего объекта. Строка состоит из 0 и 1 и её значение - двоичное представления значения объекта в обратном коде. Дробная часть содержит 4 символа и образуется путём ОТБРАСЫВАНИЯ символов после него (НЕ математическим округлением). Примеры: 
    -5.75 = "010.0011" (Незначащие 0-и в записи положительного числа и незначащие 1-цы в записи отрицательного числа отбрасываются)
    11 = "1011" (Дробная часть отбрасывается, если содержит только нули в записи положительного числа или только единицы в записи отрицательного числа)

# Краткая теория

Теория

Обратный код — метод вычислительной математики, позволяющий вычесть одно число из другого, используя только операцию сложения над натуральными числами.

В двоичной системе счисления обратный код очень просто получается инверсией каждого бита (заменой «0» на «1» и наоборот). Дополнительный код может быть получено из Обратного кода добавлением единицы в младший значащий разряд (бит).
Например, рассмотрим вычитание двух целых чисел: 01100100−11101001 . Каждое число записываем 8-битовым кодом, при этом самый старший, 8-й разряд считается знаковым.

0110 0100  (x) \
 \+             \
1110 1001  (первое дополнение y)          \
 \+          \
1  (чтобы получить второе дополнение)      \
————————————                               \
 1 0100 1110
