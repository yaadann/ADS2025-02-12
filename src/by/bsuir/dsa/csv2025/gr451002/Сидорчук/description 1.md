# Название

Спиральное преобразование матрицы

# Описание

Условие задания

**Русский:**  
Дана размерность квадратной матрицы `n` и сама матрица `n × n`. Выполните следующие операции:
1. Поверните матрицу на 90 градусов по часовой стрелке
2. Обойдите повернутую матрицу по спирали по часовой стрелке, начиная с левого верхнего угла
3. Верните результирующий массив, где элементы расположены **в точном порядке их посещения при спиральном обходе**

## Формат ввода 

- Первая строка: целое число `n` (размерность матрицы)
- Следующие `n` строк: по `n` целых чисел в каждой (элементы матрицы) через пробел

# Формат вывода

-Одна строка с `n²` целыми числами через пробел - элементы в порядке спирального обхода 

# Краткая теория

Матрицы: поворот и спиральный обход

## Определение

**Матрица** — это прямоугольная таблица чисел, arranged in rows and columns. В контексте данной задачи мы работаем с **квадратными матрицами** размера $n \times n$.

## Основные операции

### Поворот матрицы

Поворот матрицы на 90° по часовой стрелке — это линейное преобразование, которое можно выразить через индексы:

$$
\text{rotated}[j][n-1-i] = \text{matrix}[i][j]
$$

**Математическая интерпретация:** Такой поворот эквивалентен последовательному применению двух операций:
1. Транспонирование матрицы: $A^T$
2. Отражение относительно вертикальной оси

### Спиральный обход

Спиральный обход матрицы — это способ traversing элементов по концентрическим "слоям" от границы к центру.

**Алгоритмический подход:** Используем четыре граничных указателя:
- `top`, `bottom` — границы по строкам
- `left`, `right` — границы по столбцам

Обход выполняется в четыре этапа на каждой итерации:
1. Слева направо по верхней строке
2. Сверху вниз по правому столбцу  
3. Справа налево по нижней строке
4. Снизу вверх по левому столбцу

После каждого полного обхода границы сужаются к центру.

## Пример

**Исходная матрица:**
[1, 2, 3]
[4, 5, 6]
[7, 8, 9]

**После поворота на 90° по часовой стрелке:**
[7, 4, 1]
[8, 5, 2]
[9, 6, 3]
