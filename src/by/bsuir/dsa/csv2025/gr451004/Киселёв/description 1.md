# Название

Навигатор для складских роботов 

# Описание

Дана прямоугольная сетка (склад) размером N×M, состоящая из клеток четырех типов:

· свободные клетки ('.')
· препятствия ('#')
· стартовая позиция робота ('R')
· целевая позиция ('C')

Робот может перемещаться на одну клетку вверх, вниз, влево или вправо за один шаг, но не может:

· выходить за границы сетки
· проходить через препятствия

Требуется: найти минимальное количество шагов, за которое робот может добраться от стартовой позиции 'R' до целевой позиции 'C'. Если такой путь невозможен, вернуть -1.

# Краткая теория

Задание: Навигатор для складских роботов

Категория
**Графы → Обходы графов (BFS)**

Сложность
Средняя

Описание задачи

Вы разрабатываете систему навигации для автоматизированного склада. Роботы-погрузчики перемещаются по складу, который представлен в виде двумерной сетки размером `N × M`.

Обозначения на карте:
- `'.'` — свободная проходимая ячейка
- `'#'` — препятствие (стеллаж, который нельзя обойти)  
- `'R'` — стартовая позиция робота
- `'C'` — целевая ячейка, куда нужно доставить груз

Робот за один ход может переместиться на одну клетку **вверх, вниз, влево или вправо**. Нельзя выходить за границы сетки и перемещаться через препятствия.

Задача

Напишите функцию, которая определяет **минимальное количество шагов**, необходимое роботу, чтобы добраться от старта `'R'` до цели `'C'`.

**Если путь невозможен, верните `-1`.**

Формат входных данных
- Двумерный список (матрица) символов `List[List[str]]`
- Гарантируется, что в сетке присутствует ровно один `'R'` и хотя бы один `'C'`

Формат выходных данных
- Целое число — минимальное количество шагов или `-1`

Пример 1

**Вход:**
```python
grid = [
    ['.', '.', '.', '#', '.'],
    ['.', '#', '.', '#', '.'],
    ['.', '#', '.', '.', '.'],
    ['R', '.', '#', '#', 'C'],
    ['.', '.', '.', '.', '.']
]
