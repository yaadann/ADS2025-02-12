# Расширенное сравнение по модулю

Дано три целых числа `a`, `b`, `m`. Требуется определить, существует ли целое число `x`, такое что выполняется сравнение:

\[
a \cdot x \equiv b \pmod{m}
\]

Если решение существует, выведите **любое** такое `x` в диапазоне от `0` до `m-1`.  
Если решения нет — выведите `NO`.

### Вход
Три целых числа:  
`a` `b` `m`, где  
- \(1 \le m \le 10^9\)  
- \(0 \le a, b \le 10^9\)

### Выход
Одно число `x` от `0` до `m-1`, или строку `NO`.

### Пример
Ввод:

## Краткая теория

Уравнение

\[
a x \equiv b \pmod{m}
\]

имеет решение тогда и только тогда, когда:

\[
\gcd(a, m) \mid b.
\]

Пусть \(d = \gcd(a, m)\). Тогда уравнение можно упростить, поделив на \(d\):

\[
\frac{a}{d} x \equiv \frac{b}{d} \pmod{\frac{m}{d}}.
\]

Поскольку \(\gcd(a/d, m/d) = 1\), существует обратный элемент:

\[
x \equiv \frac{b}{d} \cdot (a/d)^{-1} \pmod{m/d}.
\]

Для нахождения обратного элемента используется расширенный алгоритм Евклида.
