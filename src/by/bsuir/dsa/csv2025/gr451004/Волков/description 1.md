# Название

# Описание

Обход графа в ширину

Дан невзвешенный ориентированный граф в виде списка смежности и номер стартовой вершины. Ваша задача — обойти граф в ширину (BFS) и вывести вершины в порядке их посещения.

### Формат входных данных
- Первая строка: `n m s` (количество вершин, рёбер и стартовая вершина)
- Следующие `m` строк: `u v` (ребро из вершины `u` в вершину `v`)

### Формат выходных данных
- Вершины в порядке обхода BFS, начиная с `s`, разделенные пробелом

### Ограничения
- 1 ≤ n ≤ 10^5
- 0 ≤ m ≤ 2×10^5
- Временная сложность: O(n + m)

# Краткая теория

## Алгоритм BFS

1. Создаем очередь и массив посещенных вершин
2. Добавляем стартовую вершину в очередь и помечаем посещенной
3. Пока очередь не пуста:
   - Извлекаем вершину из очереди
   - Обрабатываем ее
   - Добавляем всех непосещенных соседей в очередь

**Сложность:** O(V + E)
**Память:** O(V)
