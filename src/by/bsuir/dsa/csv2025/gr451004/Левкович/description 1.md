# Название

Сортировка подсчётом

# Описание

Сортировка подсчетом

## Формальная постановка
Дан массив целых чисел в диапазоне от 0 до k. Отсортировать массив за линейное время.

## Реальная мотивация
Сортировка подсчетом широко используется в системах, где нужно быстро отсортировать данные с ограниченным диапазоном значений: обработка оценок студентов, анализ возрастных групп, статистическая обработка дискретных данных.

## Формат ввода
Первая строка: целое число n (1 ≤ n ≤ 1000) - количество элементов  
Вторая строка: n целых чисел в диапазоне [0, 100], разделенных пробелами

## Формат вывода
Отсортированный по возрастанию массив

## Пример
Ввод:
6
4 2 2 8 3 3

text

Вывод:
2 2 3 3 4 8

# Краткая теория

Алгоритм сортировки подсчетом

## Математическая основа
Для массива $A$ длины $n$ с элементами в диапазоне $[0, k]$:

1. **Подсчет частот:** Создаем массив счетчиков $C$ размером $k+1$
   $$ C[i] = \text{количество элементов равных } i $$

2. **Префиксные суммы:** Преобразуем $C$ в массив позиций
   $$ C[i] = C[i] + C[i-1] \quad \text{для } i = 1 \text{ до } k $$

3. **Построение результата:** Размещаем элементы в выходном массиве $B$
   $$ B[C[A[j]] - 1] = A[j] $$
   $$ C[A[j]] = C[A[j]] - 1 $$

## Псевдокод
Counting-Sort(A, k):
C = [0] * (k+1)
for j = 1 to length(A):
C[A[j]] = C[A[j]] + 1
for i = 1 to k:
C[i] = C[i] + C[i-1]
for j = length(A) downto 1:
B[C[A[j]]] = A[j]
C[A[j]] = C[A[j]] - 1

## Сложность
- **Время:** $O(n + k)$
- **Память:** $O(n + k)$

## Ограничения
Работает только для целых чисел с ограниченным диапазоном
