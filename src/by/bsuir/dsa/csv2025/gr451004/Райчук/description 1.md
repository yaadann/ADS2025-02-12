# Название

Количество непересекающихся вхождений

# Описание

Количество непересекающихся вхождений.
Даны строки S (текст) и P (подстрока).
Подсчитать количество непересекающихся вхождений P в S.

# Краткая теория

Алгоритм Кнут-Морриса-Пратта (KMP) позволяет искать подстроку за время O(n + m), где
n - длина строки S,
m - длина P.
Основные элементы алгоритма:
LPS-массив (Longest Prefix Suffix) - для каждой позиции хранит длину максимального префикса P, совпадающего с её суффиксом.
При несоответствии символов алгоритм не возвращается назад в S, а использует LPS, чтобы продолжить поиск быстрее.
Чтобы считать непересекающиеся вхождения, после полного совпадения длиной m нужно откатиться не по LPS, а сразу продолжить с позиции после совпадения.
