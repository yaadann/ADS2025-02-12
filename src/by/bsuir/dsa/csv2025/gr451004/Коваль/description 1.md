# Название

Рюкзак

# Описание

Рюкзак
Вам дан рюкзак грузоподъёмностью W и список из n предметов. Каждый i-й предмет имеет вес weight[i] и стоимость value[i].
Ваша задача — определить максимальную суммарную стоимость предметов, которые можно унести в рюкзаке, при условии, что нельзя взять более одного экземпляра каждого предмета (задача «0-1»).
Формат входных данных:
Первая строка: целое число W (грузоподъёмность рюкзака).
Вторая строка: целое число n (количество предметов).
Следующие n строк: два целых числа, разделённых пробелом — weight[i] и value[i] соответственно.
Формат выходных данных:
Одно целое число — максимальная суммарная стоимость.

# Краткая теория

Задача о рюкзаке
Задача о рюкзаке является классической проблемой в комбинаторной оптимизации и прекрасно решается с помощью динамического программирования.
Идея решения
Создадим двумерный массив dp, где:
dp[i][w] — максимальная стоимость, которую можно получить, используя первые i предметов и имея рюкзак вместимостью w.
Рекуррентные соотношения
Базовый случай:
Если нет предметов или нулевая вместимость: dp[0][w] = 0 и dp[i][0] = 0.
Переход:
Если вес текущего предмета weight[i-1] больше текущей вместимости w, то мы не можем его взять:
dp[i][w] = dp[i-1][w]
Иначе, мы выбираем максимум между вариантом взять предмет и не взять его:
dp[i][w] = max(dp[i-1][w], dp[i-1][w - weight[i-1]] + value[i-1])
Временная сложность
Время: O(n * W), где n — количество предметов, W — вместимость рюкзака.
Память: O(n * W), которую можно оптимизировать до O(W).
