# Название

Поиск уникальных слов с помощью хэш-таблицы

# Описание

Дана строка, состоящая из слов, разделённых пробелами. Требуется определить количество уникальных слов в этой строке.

Используйте хэш-таблицу (или аналогичную структуру данных с хешированием), чтобы эффективно выявить повторяющиеся слова.

Формат ввода:
Одна строка текста.

Формат вывода:
Одно целое число — количество уникальных слов.

Пример:
Ввод:
апельсин яблоко яблоко груша апельсин
Вывод:
3

# Краткая теория

Хеширование — это способ преобразования данных (например, строки) в числовое значение фиксированной длины, называемое хеш-значением.
Структура данных, использующая хеширование, — хеш-таблица, позволяет быстро выполнять операции поиска, добавления и проверки наличия элемента.

Основная идея:

хеш-функция вычисляет хеш ключа;

по хешу определяется позиция элемента в памяти;

благодаря этому доступ к данным обычно выполняется за O(1).

Хеш-таблицы часто применяют для проверки уникальности элементов или для подсчёта их количества.
