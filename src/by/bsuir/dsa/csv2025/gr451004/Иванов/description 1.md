# Описание

В стране есть `n` городов, соединённых `m` двусторонними каналами связи. Каждый канал связывает два города и позволяет передавать информацию в обоих направлениях.

Служба безопасности обеспокоена надёжностью коммуникационной сети. Они хотят выявить все **критические каналы связи** — такие каналы, при выходе из строя которых хотя бы один город окажется отрезанным от остальной сети, либо сеть распадётся на несколько изолированных частей.

Другими словами, критический канал — это единственный путь между некоторыми городами. Если такой канал перестанет работать, связь между этими городами будет полностью потеряна.

Помогите службе безопасности составить список всех критических каналов. Каждый канал представлен парой городов `u` и `v`. Выведите все критические каналы в порядке возрастания номеров городов (если канал соединяет города `u` и `v`, где `u < v`, выведите `u v`).

**Мост** (bridge) — это такое ребро неориентированного графа, удаление которого увеличивает количество компонент связности графа. Другими словами, мост — это единственный путь между двумя частями графа.

#### Основные свойства мостов:
- В дереве все рёбра являются мостами
- В цикле нет мостов
- Если граф содержит цикл, то рёбра этого цикла не являются мостами
- Мост всегда принадлежит какому-либо простому пути

#### Алгоритм поиска мостов (DFS)

Для поиска мостов используется модификация алгоритма обхода в глубину (DFS) с использованием двух массивов:
- `tin[v]` — время входа в вершину `v` при DFS
- `low[v]` — минимальное время входа среди всех вершин, достижимых из `v` по рёбрам дерева DFS и одному обратному ребру

**Критерий моста:** Ребро `(u, v)` является мостом тогда и только тогда, когда `low[v] > tin[u]` (или `low[u] > tin[v]`).

#### Временная сложность
- **O(V + E)**, где V — количество вершин, E — количество рёбер
- Алгоритм выполняет один обход графа в глубину

#### Применение
- Анализ надёжности сетей связи
- Поиск уязвимых мест в инфраструктуре
- Построение двусвязных компонент графа
