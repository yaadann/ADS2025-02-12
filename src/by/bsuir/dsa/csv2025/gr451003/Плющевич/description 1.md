## Название

Почти-палиндромная подстрока

## Описание

Дана строка `text` и число `k_len`. Найдите все стартовые индексы `i` такие, что подстрока длины `k_len`  
(`text[i..i+k_len-1]`) может стать палиндромом, если изменить **не более одного символа**.  
Вернуть индексы в порядке возрастания.

## Примеры

text = "abca"  
k_len = 4  
Ответ: [0]  
Объяснение: "abca" → меняем `b` на `c` → "acca", палиндром.

text = "abcdefg"  
k_len = 3  
Ответ: [0, 1, 2, 3, 4]

## Ограничения

0 <= |text| <= 2e5  
1 <= k_len <= |text|  
Символы: латиница a..z (нижний регистр)

## Теория

Пусть подстрока S = text[i..i+m-1], где m = k_len.

Подстрока является почти-палиндромом, если количество несовпадений на симметричных позициях  
не превышает 1 — то есть число позиций j, для которых:

S[j] != S[m - 1 - j]

не больше одного.

## Проверка

- Использовать два указателя: left = 0, right = m - 1
- Идти навстречу, сравнивая символы
- Считать несовпадения
- Если несовпадений ≤ 1 — индекс i является подходящим

## Сложность

Время: O(n * k_len)  
Память: O(1)
