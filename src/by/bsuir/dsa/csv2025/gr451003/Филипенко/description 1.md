# Название

Кратчайший путь в графе

# Описание

Дан ориентированный граф с весами на рёбрах. Требуется найти кратчайший путь от вершины `s` до вершины `t`.  
Формат входных данных:  
- Первая строка: `n m` — количество вершин и рёбер.  
- Следующие `m` строк: `u v w` — ребро из `u` в `v` весом `w`.  
- Последняя строка: `s t` — стартовая и конечная вершины.  

Формат выходных данных:  
- Одно число — длина кратчайшего пути, либо `-1`, если путь не существует.

# Краткая теория

Для поиска кратчайшего пути в графе с неотрицательными весами используется алгоритм Дейкстры.  
Его идея: пошагово расширять множество вершин с известным минимальным расстоянием, выбирая на каждом шаге вершину с наименьшей текущей меткой.

Формально:
- Инициализируем расстояния: `dist[s] = 0`, остальные `∞`.
- Используем приоритетную очередь для выбора вершины с минимальным расстоянием.
- Для каждой вершины пересчитываем расстояния до соседей: 
  dist[v] = min(dist[v], dist[u] + w)
