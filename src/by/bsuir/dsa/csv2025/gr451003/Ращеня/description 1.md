# Название

Анализ надежности моста

# Описание

Численные методы. Численное интегрирование - Анализ надежности моста

## Задача: "Анализ надежности моста под динамической нагрузкой"

Необходимо разработать программу для анализа динамической нагрузки на пешеходный мост с использованием методов численного интегрирования
и алгоритмов обработки данных.
Инженерам требуется оценить надежность пешеходного моста под воздействием динамической нагрузки. Известно, 
что при прохождении группы людей возникает периодическая нагрузка, описываемая функцией:

f(t) = A * sin(ωt) + B * e^(-γt) * cos(Ωt) + C * t

**Параметры:**
> A (5000 H) - амплитуда основной нагрузки
> B (2000 H) - амплитуда затухающих колебаний
> C (100 Н/с) - линейный рост нагрузки
> ω (2pi рад/с) - основная частота
> γ (0.1 с^(-1)) - коэффициент затухания
> Ω (4pi рад/с) - частота затухающих колебаний

## Требуется реализовать:

1. **Численное интегрирование методом Симпсона**
   - Реализовать функцию для вычисления интеграла нагрузки на интервале [0, n] секунд
   - Вычислить среднюю нагрузку за период наблюдения

2. **Анализ максимальной нагрузки**
   - Найти максимальное значение нагрузки на интервале

## Вводимые данные:
- Нижняя граница интегрирования
- Верхняя граница интегрирования
- Количество подинтервалов для метода Симпсона
- Шаг дискретизации для поиска максимума

## Выходные данные
Программа должна выводить:
- Среднюю нагрузку
- Максимальную нагрузку

## Численное интегрирование

### Метод Симпсона (Парабол)
Основан на приближении функции квадратичными параболами на каждом отрезке разбиения.
Окончательная формула выглядит следующим образом:

∫(от a до b) f(x) dx ≈ (h / 3) * [f(x₀) + f(xₙ) + 4*(f(x₁) + f(x₃) + ... + f(xₙ₋₁)) + 2*(f(x₂) + f(x₄) + ... + f(xₙ₋₂))]

> n - чётное число отрезков
> h = (b - a)/n - шаг
> x₀ = a, x₁ = a + h, x₂ = a + 2h, ..., xₙ = b - узлы сетки

#### Алгоритм применения 

1. Задайте функцию f(x), пределы интегрирования a и b, и чётное число разбиений n.
2. Вычислите шаг h = (b - a)/n.
3. Создайте сетку узлов: x_i = a + i*h, где i = 0, 1, 2, ..., n и вычислите значения функции в этих узлах.
4. Подставьте значения в формулу Симпсона:

sum_odd = f(x₁) + f(x₃) + ... + f(xₙ₋₁)
sum_even = f(x₂) + f(x₄) + ... + f(xₙ₋₂)
integral = (h / 3) * [f(x₀) + f(xₙ) + 4*sum_odd + 2*sum_even]
