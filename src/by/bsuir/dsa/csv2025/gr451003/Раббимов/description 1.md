# Описание

Найти все подстроки по заданным критериям

## Задание

Дана непустая строка, содержащая любые символы в любой последовательности. Необходимо найти все подстроки, удовлетворяющие следующим требованиям:

1. Состоят либо только из кириллических символов, либо только из латинских
2. Минимальная длина подстроки: 3 символа
3. Максимальная длина подстроки: 15 символов
4. Подстрока начинается с заглавного символа
5. В подстрока содержит только один заглавный символ

> **Замечания:** 
> - составить массив с подстроками;
> - вывести все подстроки через пробел в порядке, в котором они появляются в заданной строке;

## Пример

- Ввод: 76МАРтшщ778*впрвы+-*Dfsr65565dggd8778Hik
- Ожидаемый вывод: Ртщщ Dfsr Hik

## Поиск подстроки

### Определение и особенности

**Подстрока** - непрерывная последовательность символов внутри исходной строки.

Классификация символов:
1. Кириллические символы:
    > Заглавные: А-Я, Ё
    > Строчные: а-я, ё
2. Латинские символы:
    > Заглавные: A-Z
    > Строчные: a-z
Алгоритмические подходы:

    1. Полный перебор - проверка всех возможных подстрок длины от 3 до 15

    2. Скользящее окно - эффективное перемещение по строке с проверкой условий

    3. Конечные автоматы - последовательная обработка символов с отслеживанием состояния

Ключевые проверки:

    1. Определение алфавита (кириллица/латинска)
    2. Проверка регистра символов
    3. Подсчет заглавных символов
    4. Валидация длины подстроки

Регулярные выражения:

Для данной задачи можно использовать регулярные выражения для эффективной проверки условий:

    [А-ЯЁ][а-яё]{2,14} - кириллические подстроки

    [A-Z][a-z]{2,14} - латинские подстроки
