# Название

Поиск пар предок-потомок в дереве

# Описание

Задание
 
Дана непустая последовательность групп символов, разделённых пробелами. Каждая группа задаёт некоторый двоичный код. Также даны целые числа K и D.

Для данной последовательности:

1. Построить двоичное дерево поиска, где ключами являются двоичные коды (сравнение лексикографическое).
2. Для каждой пары вершин (A, B), где A - предок B, вычислить XOR значений на пути от A к B и вывести все такие пары с их XOR значениями.
3. Найти количество таких пар вершин, где:
   - результат XOR операции содержит ровно K единичных битов
   - расстояние между A и B в дереве не превышает D
   - сумма количеств единичных битов в кодах A и B является простым числом
4. Вывести количество подходящих пар и сами пары.

> **Замечания:**
> - расстояние между вершинами измеряется количеством рёбер на пути между ними;
> - путь включает все вершины от A до B включительно;
> - 1 не считается простым числом;
> - пары выводятся в формате (A,B):XOR.

## Пример

- Ввод:<br>
`0001 1000 1010 0010 0101 2 3`<br>
- Ожидаемый вывод:<br>
`1`<br>

**Объяснение:**
- Все пары предок-потомок с их XOR значениями
- Подходящие пары: (1,2) и (1,8), так как:
  - XOR содержит ровно 2 единицы (3=11₂, 9=1001₂)
  - Расстояние ≤ 3
  - Сумма битов A и B простая (1+1=2, 1+1=2)


# Краткая теория

## Двоичное дерево поиска

Двоичное дерево поиска — структура данных с упорядоченными ключами, где для каждого узла:
- Все ключи в левом поддереве меньше ключа узла
- Все ключи в правом поддереве больше ключа узла

## Побитовый XOR

XOR (исключающее ИЛИ) — битовая операция:
- 0 XOR 0 = 0
- 0 XOR 1 = 1
- 1 XOR 0 = 1
- 1 XOR 1 = 0

**Свойства:**
- Ассоциативность: (a XOR b) XOR c = a XOR (b XOR c)
- a XOR a = 0
- a XOR 0 = a

## Простые числа

Простое число — натуральное число больше 1, которое имеет ровно два делителя: 1 и само себя.

## Алгоритм решения

1. Построение BST из двоичных кодов
2. Рекурсивный обход для нахождения всех пар предок-потомок
3. Вычисление XOR пути для каждой пары
4. Проверка условий: количество битов в XOR, расстояние, простота суммы битов
