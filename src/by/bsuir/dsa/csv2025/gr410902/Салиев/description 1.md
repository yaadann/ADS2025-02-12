# Краткая теория

## Условие задачи

ФИО (String name)

Средний балл (double gpa)

Количество пропусков занятий (int absences)

Номер группы (String group
- Массив содержит не более \( k \) *инверсий* — таких пар индексов \( (i, j) \), что \( i < j \), но \( a[i] < a[j] \).

Требуется:

Создаёт массив из нескольких студентов (не меньше 6–7, чтобы сортировка была осмысленной).
Предлагает пользователю на выбор критерий сортировки:
По ФИО (в алфавитном порядке, A → Z)
По среднему баллу (по убыванию, от большего к меньшему)
По количеству пропусков (по возрастанию, от меньшего к большему)
По номеру группы (в алфавитном порядке)
Сортирует массив строго алгоритмом пузырьковой сортировки, но в форме двунаправленного пузырька (cocktail shaker sort):
Сначала проход слева направо — крупные элементы «всплывают» в конец.
Затем проход справа налево — маленькие элементы «опускаются» в начало.
Реализует оптимизацию: если за полный цикл (проход туда-обратно) не было ни одной перестановки, сортировка завершается.
Запрещено использовать:
Arrays.sort, Collections.sort, List.sort
Comparator, лямбда-выражения для сортировки
Разрешено только свой пузырёк.
После сортировки программа выводит отсортированный список студентов.
