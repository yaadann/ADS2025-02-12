# Название

Светофоры на улице

# Описание


В городе строят новую улицу с магазинами, и вдоль неё планируют установить светофоры. Каждый светофор обеспечивает безопасную зону пешеходного перехода в виде отрезка вдоль улицы (ось X). Пешеход может безопасно идти только в зоне, покрытой хотя бы одним светофором.

Улица представлена одной прямой (ось X). Каждый светофор задаёт начало и конец безопасной зоны [l, r]. Задача — посчитать общую длину безопасной зоны (не считая пересекающиеся зоны дважды).

Если зоны светофоров пересекаются, длина пересечения учитывается только один раз. В решении нужно использовать алгоритм сканирующей линии (sweep line).

## Вход
Первое число N — количество светофоров (1 ≤ N ≤ 100000).  
Далее N строк, каждая с двумя числами l r (координаты зоны светофора, l < r, целые числа, могут быть отрицательными).

## Выход
Одно число — общая длина безопасной зоны.

## Пример
Input:  
3  
1 5  
2 6  
8 10  

Output:  
7


# Краткая теория

Метод сканирующей прямой (англ. scanline) заключается в сортировке точек на координатной прямой либо каких-то абстрактных «событий» по какому-то признаку и последующему проходу по ним.

Он часто используется для решения задач на структуры данных, когда все запросы известны заранее, а также в геометрии для нахождения объединений фигур.
