# Название

Анализ социальной сети

# Описание

Анализ социальной сети

## Условие задачи
Разработайте программу для анализа социальной сети, представленной в виде графа. Программа должна определять размер наибольшего сообщества, где все участники непосредственно связаны друг с другом (каждый с каждым). 

Граф задаётся матрицей смежности, где:
- 1 означает наличие связи между пользователями
- 0 означает отсутствие связи

Требуется найти размер максимальной клики - наибольшей группы пользователей, где все попарно связаны.

```java
Вход : "4 0 1 1 0 1 0 1 1 1 1 0 0 0 1 0 0"
Выход : "3"
```

# Краткая теория

## Теоретическая справка
**Клика** в теории графов - это подмножество вершин графа, такое что любые две вершины в этом подмножестве соединены ребром. 

**Максимальная клика** - это клика, которая не может быть расширена добавлением другой вершины.

**Матрица смежности** - квадратная матрица, где элемент a[i][j] показывает наличие ребра между вершинами i и j.

Для неориентированного графа матрица смежности является симметричной относительно главной диагонали.

**Поиск в глубину** (англ. depth-first search, DFS) или эйлеровым обходом называется рекурсивный алгоритм обхода корневого дерева или графа, начинающий в корневой вершине (в случае графа может быть выбрана произвольная вершина) и рекурсивно обходящий весь граф, посещая каждую вершину ровно один раз.
