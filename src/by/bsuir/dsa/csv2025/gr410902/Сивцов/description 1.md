# Описание

Оптимизация маршрута доставки

### Сведения

Вы работаете в сервисе доставки и вам нужно оптимизировать маршрут курьера. Курьер выезжает со склада (точка 0), должен развезти заказы по $n$ адресам (точки $1..n$) и вернуться обратно на склад. Найдите самый короткий маршрут, который проходит через все адреса доставки по одному разу.

### Что дано

* $n$ - количество адресов доставки (от 1 до 15)
* Таблица расстояний между всеми точками (склад + адреса доставки)

### Что найти

* Минимальную длину маршрута от склада через все адреса и обратно на склад


# Краткая теория

## Краткая теория
Задача коммивояжера (TSP - Travelling Salesman Problem) является классической NP-трудной задачей комбинаторной оптимизации. Для ее решения используется метод динамического программирования с битовыми масками.

### Основная идея алгоритма

* Состояние $dp$[mask][last] хранит минимальное расстояние для посещения всех точек из маски $mask$, заканчивая в точке $last$

* Маска представляет битовый вектор, где $i$-й бит равен 1, если точка $i$ уже посещена

* Начальное состояние: посещен только склад (mask = 1, last = 0)

* Переходы: из текущего состояния (mask, last) переходим во все непосещенные точки $next$

* Ответ: минимальное значение $dp$[full_mask][last] + $distance$[last][0] для всех $last$

* Сложность: $O(2ⁿ × n²)$, что эффективно при $n ≤ 15$
