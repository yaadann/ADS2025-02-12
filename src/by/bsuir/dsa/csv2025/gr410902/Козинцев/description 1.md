# Название

 Расчет критической температуры химического реактора методом Ньютона

# Описание

Расчет критической температуры химического реактора

Для безопасной эксплуатации промышленного реактора необходимо определить критическую температуру, при которой происходит изменение режима протекания химической реакции. Эта температура является корнем кубического уравнения:

$f(T) = aT^3 + bT^2 + cT + d = 0$

где коэффициенты `a, b, c, d` зависят от конкретных параметров реакции.

## Технические требования

### Входные данные
Метод принимает строку с шестью значениями через пробел:
- `a, b, c, d` - коэффициенты уравнения
- `T₀` - начальное приближение температуры
- `N` - максимальное число итераций

### Выходные данные
Для корректной проверки метод должен возвращать строку, содержащую **целое число** - результат умножения найденной температуры на **10000** (с округлением до целого).

## Пример

Для уравнения: $T^3 - 2T^2 + 3T - 5 = 0$

```java
// Входные данные
String input = "1.0 -2.0 3.0 -5.0 2.0 5";

// Выходные данные  
String output = "18437";  // 1.8437 × 10000 = 18437
```
## Примечание
Для решения использовать метод Ньютона с итерационной формулой:

$T_{n+1} = T_n - \frac{f(T_n)}{f'(T_n)}$

# Метод Ньютона (Метод касательных)

**Метод Ньютона** — это **итерационный численный метод**, предназначенный для нахождения корней (нулей) нелинейной функции $f(x)$. Он является одним из самых эффективных методов, при условии удачного выбора начального приближения.

## Принцип работы

Принцип основан на использовании **касательной** к графику функции в текущей точке $x_n$. Новое, более точное приближение $x_{n+1}$ находится как точка пересечения этой касательной с осью $OX$. 

## Ключевая формула

Итерационный процесс описывается следующей формулой:

$$x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}$$

Где:
* $x_n$ — текущее приближение корня.
* $f(x_n)$ — значение функции в этой точке.
* $f'(x_n)$ — значение **производной** функции в этой точке.


## Сходимость

Метод Ньютона обладает **квадратичной сходимостью** (Quadratic Convergence), что означает, что при удачном начальном приближении он сходится очень быстро.
