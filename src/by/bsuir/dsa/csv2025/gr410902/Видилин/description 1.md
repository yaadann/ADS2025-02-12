# Краткая теория

## Условие задачи

Дан массив целых чисел \( a \) длины \( n \).  
Известно, что массив почти отсортирован по убыванию в следующем смысле:

- Массив содержит не более \( k \) *инверсий* — таких пар индексов \( (i, j) \), что \( i < j \), но \( a[i] < a[j] \).

Требуется:

1. **Отсортировать массив по убыванию**, используя модифицированную сортировку вставками.
2. **Ускорить сортировку вставками**, заменив линейный поиск позиции вставки на бинарный поиск.
3. Реализовать алгоритм:
   - бинарный поиск позиции в отсортированном убывающем префиксе
   - сдвиг элементов обычным циклом
4. Оценить асимптотику:
   - в худшем случае
   - в терминах количества инверсий \( k \)

Программа должна:

- считать \( n \) и массив из \( n \) чисел
- отсортировать его бинарной сортировкой вставками
- вывести результат
